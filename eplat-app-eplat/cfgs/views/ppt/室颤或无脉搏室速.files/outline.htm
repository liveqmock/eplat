3
学习目的
  • ACLS操作要点.
  • 了解第一阶段的ABCD 通常是对无意识、无呼吸、无脉搏的病人所进行的评估和最初治疗（CPR和首次除颤） 。
  • 对于初次除颤不成功的病人，应立即给予第二阶段的ABCD治疗，即进行气管插管以提供有效通气、继续胸外按压、开放静脉通道，再次除颤。
4
学习目的
  • 一个成功的ACLS提供者应该做到：
  • 在病人出现室颤或无脉搏室速情况的10分钟内快速给予第一、二阶段的ABCD治疗
  • 在条件不具备的条件下立即CPR
  • 在条件许可的情况下使用AED或除颤器
  • 如果有多个ACLS提供者可以组成一个抢救小组：
    • 第二个人可以帮助继续进行 CPR
    • 第三个负责气管插管，保持呼吸畅通
    • 第四个人负责静脉给药
  • 适当选用肾上腺素、抗心律失常或其它药物

第一阶段 ABCD
  • 重点:基本 CPR和除颤
  • A : 开放气道
  • B : 检查呼吸，进行人工呼吸
  • C : 检查脉搏，进行胸外按压
  • D : 对室颤/无脉搏室速病人进行评估及给予除颤治疗

除颤方面有什么新内容？
  • 双向波形除颤：最为肯定
  • 不同的波形：可接受
  • 最常见的：单相波 (DpSn) (A)
  • 极少用：单相波 (TrEx)
  • 多样新形式：双相波 (B and C)
  • 所有这些都被广泛接受
  • 新波形：如经人体临床验证可用

第二阶段 ABCD
  • A ：尽快进行气管插管
  • B ：通过听诊确定正确的位置
  • B ：判断正确位置的第二种方法
    • 终末潮气 CO2
    • 食道探测器检查
  • B ：预防气管插管移位
    • 使用插管固定器
    • 被证实的结带或其它技术
  • B ：检测氧气流量与通气情况

室颤/无脉搏室速:
自主循环的恢复
  • 让第二阶段ABCD准则引导后续治疗
    • A = 保护气道，维持开通状态
    • A = 在转运中稳定插管，避免滑脱
    • B = 检查通气 (CO2) 和氧饱和度 (O2)
    • C = 监测心律； 给予适合的药物
    • D = 使用抗心律失常药物；如除颤，则再使用同类型药物
    • C = 维持血压和心跳： 用多巴胺或多巴酚丁胺       (避免用肾上腺素、异丙肾上腺素、去甲肾上腺素)
