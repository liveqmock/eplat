技术目标
  • 开放气道并维持通气，因为适当通气是在鉴别可逆因素同时对心室停搏采取的主要抢救措施。

第一步ABCD步骤
  • A = 气道。
  • B = 呼吸。检查呼吸，提供正压通气
  • C = 循环。检查循环，给予胸外按压。
  • D = 除颤。判断和对室颤或无脉搏室速给予除颤。

  • B = 呼吸。进一步证实开放气道器械置入正确
  •        （潮气末二氧化碳浓度、食道探测器）
